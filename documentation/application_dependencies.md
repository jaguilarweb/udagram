
# Dependencies


```
- Node v11.15.0 (LTS) or more recent. While older versions can work it is advisable to keep node to latest LTS version

- npm 6.7.0 (LTS) or more recent

- AWS CLI v2, v1 can work but was not tested for this project

- A RDS database running Postgres.

- A S3 bucket for hosting uploaded pictures.

```
